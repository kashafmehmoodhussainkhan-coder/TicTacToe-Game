# TicTacToe-Game
A fun Tic Tac Toe game coded in C++
This project demonstrates the use of object-oriented programming concepts such as classes, functions, arrays, and loops to build a simple yet engaging console-based game.
# Features
- Player vs Player mode
- 3x3 game board displayed in the console
- Players take turns entering their move (row and column)
- First player uses X, second player uses O
- Automatic detection of win or draw conditions
- Clean and modular code structure using classes and functions
# Technologies Used
- Language: C++
- Concepts: Classes, Functions, Arrays, Loops
- Compiler: Any standard C++ compiler (g++, MinGW, MSVC)
- Environment: Runs in terminal/console
# How to Play
- The board is a 3x3 grid.
- Players take turns entering their move by specifying row and column numbers.
- Player 1 uses X, Player 2 uses O.
- The game ends when:
- A player wins by forming a line (row, column, or diagonal).
- The board is full (draw).
